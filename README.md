# Security-Analyst-Toolkit
A curated list of cybersecurity tools for OSINT, malware analysis, threat intelligence, phishing detection, network reputation checks, and privacy testing.


### 🔐 Data Breach Check

- **Have I Been Pwned** – Checks whether your data has ever appeared in a breach  
    [https://haveibeenpwned.com](https://haveibeenpwned.com)
    
- **DeHashed** – Searches billions of leaked credentials  
    [https://www.dehashed.com](https://www.dehashed.com)
    
- **Hudson Rock** – Checks whether your device appears in infostealer logs  
    [https://www.hudsonrock.com](https://www.hudsonrock.com)
    

---

### 🦠 Malware Analysis and Sandboxes

- **MalwareBazaar** – Safely analyzes real malware samples  
    [https://bazaar.abuse.ch](https://bazaar.abuse.ch)
    
- **VirusTotal** – Scans a file or URL with 70+ antivirus engines  
    [https://www.virustotal.com](https://www.virustotal.com)
    
- **ANY.RUN** – Runs suspicious files in a sandbox environment  
    [https://any.run](https://any.run)
    
- **Hybrid Analysis** – Executes malware and reveals its behavior  
    [https://www.hybrid-analysis.com](https://www.hybrid-analysis.com)
    

---

### 🎣 Phishing and Malicious URLs

- **URLhaus** – Tracks URLs that distribute malware  
    [https://urlhaus.abuse.ch](https://urlhaus.abuse.ch)
    
- **PhishTank** – Database of verified phishing websites  
    [https://phishtank.org](https://phishtank.org)
    
- **OpenPhish** – Active phishing URLs in real time  
    [https://openphish.com](https://openphish.com)
    

---

### 🌐 IP and Network Reputation

- **AbuseIPDB** – Blacklist of malicious IP addresses  
    [https://www.abuseipdb.com](https://www.abuseipdb.com)
    
- **GreyNoise** – Separates real attacks from background internet noise  
    [https://www.greynoise.io](https://www.greynoise.io)
    
- **Pulsedive** – Aggregates global threat intelligence feeds  
    [https://pulsedive.com](https://pulsedive.com)
    
- **IPVoid** – Checks an IP against multiple blacklists  
    [https://www.ipvoid.com](https://www.ipvoid.com)
    
- **URLVoid** – Checks a URL across multiple security lists  
    [https://www.urlvoid.com](https://www.urlvoid.com)

### 🛰️ Internet Scanning and OSINT

- **Shodan** – Finds publicly exposed devices on the internet  
    [https://www.shodan.io](https://www.shodan.io)
    
- **Censys** – Maps servers and services across the internet  
    [https://search.censys.io](https://search.censys.io)
    
- **BinaryEdge** – Analysis of the internet’s attack surface  
    [https://binaryedge.io](https://binaryedge.io)
    
- **ZoomEye** – Finds vulnerable systems across the internet  
    [https://www.zoomeye.ai](https://www.zoomeye.ai)
    
- **FullHunt** – Discovers all publicly exposed domain resources  
    [https://fullhunt.io](https://fullhunt.io)
    
- **LeakIX** – Finds misconfigured servers and exposed databases  
    [https://leakix.net](https://leakix.net)
    
- **Grayhat Warfare** – Searches public cloud storage files  
    [https://grayhatwarfare.com](https://grayhatwarfare.com)
